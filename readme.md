# DashDuino
![demo](/media/DashDuino_demo.gif?raw=true)
A simple app to read knob values from an Arduino board to a Dash app.

## Dependencies

* plotly: 4.14.3
* dash: 1.19.0
* dash_core_components: 1.15.0
  * dash_html_components: 1.1.2
  * dash_bootstrap_components: 0.12.0250
* numpy: 1.20.2
* re: 2.2.1
* serial: 3.5
