## Dependencies

The code is tested to work with these versions:

* plotly: 4.14.3
* dash: 1.19.0
  * dash_core_components: 1.15.0
  * dash_html_components: 1.1.2
  * dash_bootstrap_components: 0.12.0250
  * dash_daq: 0.5.0
* numpy: 1.20.2
* re: 2.2.1
* serial: 3.5
* json: 2.0.9
